# Product Overview

Yun Wu Portfolio - a personal portfolio website for a visual designer and photographer based in Seattle, WA.

## Purpose
Showcase creative work across three disciplines:
- Design (brand/visual design projects)
- Video (video production work)
- Photography (portrait and urban photography)

## Live Site
- Production: [yunwustudio.com](https://yunwustudio.com)
- Hosted on Cloudflare Pages with R2 storage for media

## Key Features
- Bilingual support (English/Chinese)
- Session-based splash screen with typewriter animation
- Scroll/swipe gesture navigation between splash and main content
- Photography gallery with images served from Cloudflare R2
- Admin portal at `/admin/` for photo metadata management
- Mobile-responsive design with hamburger menu
